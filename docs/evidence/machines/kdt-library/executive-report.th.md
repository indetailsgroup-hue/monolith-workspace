# คลังข้อมูลอัจฉริยะเครื่องจักร KDT — รายงานผู้บริหาร

> ณ วันที่ 2026-07-17 · แพลตฟอร์มการผลิต MONOLITH · ฉบับร่างวิจัย

## 1. วัตถุประสงค์และขอบเขต

คลังนี้รวบรวมข้อมูลเทคนิคที่อ้างอิงหลักฐานของ **เครื่องจักรงานไม้ KDT และรายการไลน์/เซลล์ 83 รายการ** เป็นแค็ตตาล็อกสองภาษา (TH/EN) สำหรับแพลตฟอร์มการผลิต MONOLITH แต่ละรายการมีโปรไฟล์เครื่องและรายงานประเมินความเข้ากันได้ ทุกค่าทางเทคนิคเป็น **Verified in documents** เท่านั้น — จากหน้าแค็ตตาล็อก งานแสดงสินค้า หรือหน้าตัวแทนจำหน่ายที่ได้รับอนุญาต — ไม่เคยเป็น **Observed in operation**

## 2. ท่าทีด้านความปลอดภัย (ต่อรองไม่ได้)

- ทุกรุ่นตั้งต้นเป็น `assessment_state = NOT_ASSESSED` และ `manufacturing_release = PROHIBITED`
- ไม่มีเอกสารใดในคลังนี้เป็นการอนุมัติการผลิตหรือการยืนยันการทำงานจริง
- ข้อมูลระดับเครื่องจริง — ฮาร์ดแวร์/เฟิร์มแวร์คอนโทรลเลอร์ เวอร์ชัน HMI/OS เส้นทางส่งไฟล์จริง ตารางทูล จุดกำเนิดพิกัด และงานตัวอย่างที่รู้ว่าดี — ยังเป็น **Unknown** จนกว่าจะมีหลักฐานหน้าเครื่อง และจะไม่ถูกอนุมานจากรุ่นพี่น้องหรือคู่มือตระกูล
- OCCT / PythonOCC ใช้เพื่อ **Interoperability & Validation เท่านั้น** ไม่ใช่ Manufacturing Authority ไฟล์ต้นฉบับห้ามแก้ไข การแปลง/heal จะได้ Versioned Derived Artifact ที่ต้องผ่านการรับงานโดยมนุษย์
- ไม่มีความลับ (IP, hostname, credential, license key) หรือเส้นทางจริงปรากฏที่ใด

## 3. สรุปความครบถ้วนของหลักฐาน

ความครอบคลุมของแหล่งข้อมูลต่อรุ่น = จำนวนกลุ่มข้อมูลเทคนิคที่มีค่าและมีแหล่งอ้างอิง / 7 (สูง = 6–7, ปานกลาง = 3–5, ต่ำ = 0–2) ค่านี้วัดการมีหลักฐาน ไม่ได้หมายถึงข้อมูลสอดคล้องกัน ผ่านการตรวจหน้าเครื่อง หรือได้รับอนุมัติการผลิต

| ระดับ | จำนวน |
|---|---|
| รายการทั้งหมด | 83 |
| หลักฐานครอบคลุมสูง (6–7/7 กลุ่มข้อมูล) | 9 |
| หลักฐานครอบคลุมปานกลาง (3–5/7 กลุ่มข้อมูล) | 61 |
| หลักฐานครอบคลุมต่ำ (0–2/7 กลุ่มข้อมูล) | 13 |
| ธงเข้าถึงแหล่งข้อมูลไม่ได้ (ซ้อนทับสามระดับข้างต้น) | 9 |

## 6. เครื่องนำร่องของ Daph

มีสองรุ่นที่เกี่ยวข้องกับเครื่องนำร่องของ Daph: **KD-610R** (เจาะหกด้าน) และ **KN-2409LP** (nesting) ทั้งคู่ถูกระบุว่าเกี่ยวข้องกับเครื่องนำร่อง แต่ยังคงเป็น **NOT_ASSESSED** และการปล่อยงานผลิตเป็น **PROHIBITED** ไม่มีการอ้างการยืนยันการทำงานจริงจากหลักฐานสาธารณะ ดู [โปรไฟล์ KD-610R](models/kd-610r/machine-profile.th.html), [ประเมิน KD-610R](models/kd-610r/compatibility-assessment.th.html), [โปรไฟล์ KN-2409LP](models/kn-2409lp/machine-profile.th.html), [ประเมิน KN-2409LP](models/kn-2409lp/compatibility-assessment.th.html)

## 7. รายการที่เข้าถึงแหล่งข้อมูลไม่ได้ (ธงซ้อนทับ)

นี่เป็นธงหลักฐานที่ซ้อนทับ ไม่ใช่ระดับความครอบคลุมลำดับที่สี่ รายการต่อไปนี้มีแหล่งข้อมูลบางส่วนเป็นภาพอย่างเดียวหรือถูกบล็อก และบางรายการอาจยังมีความครอบคลุมสูงหรือปานกลางจากแหล่งอื่น: `Batched Drill Center with servo feeding`, `KD-612KSZA`, `KE-208`, `KE-526GHTC4`, `KS-4522HLS`, `KS-543HLS`, `KS-832H`, `KS-833DP`, `The Column-mounted Four-axis Robot`.

## 8. วิธีใช้คลังนี้

- เริ่มที่ [หน้าแค็ตตาล็อก](index.html): ค้นหาด้วยรุ่น/ชื่อเรียกอื่น กรองตามตระกูล สถานะแค็ตตาล็อก ระดับหลักฐาน และ Daph-critical
- เปิด **โปรไฟล์เครื่อง** ของรุ่นเพื่อดูข้อมูลระบุรุ่น กลุ่มข้อมูลเทคนิค 7 กลุ่มพร้อมลิงก์แหล่งที่มา ข้อขัดแย้งที่บันทึกไว้ โปรไฟล์คอนโทรลเลอร์ สัญญาส่งไฟล์ คะแนนความครบถ้วน และรายการหลักฐานหน้าเครื่องที่ต้องเก็บ
- เปิด **รายงานประเมินความเข้ากันได้** เพื่อดูสถานะ NOT_ASSESSED / PROHIBITED ขอบเขตเอกสารเท่านั้น กฎรูปทรง/ขอบเขต อุปสรรค และลำดับการตรวจสอบหน้าเครื่องที่บังคับ
