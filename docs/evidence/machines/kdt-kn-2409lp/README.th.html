"เอาตามนี้เลยครับ เราต้องใส่อีกหลายเครื่อง เป็นไปไม่ได้ที่จะถ่ายหน้าเครื่องทุกเครื่องในตอนนี้

  1. ไม่มีทางลัดสู่การตัดจริง: manufacturing_release: PROHIBITED คงอยู่จนผ่าน gate E ครบ และ
