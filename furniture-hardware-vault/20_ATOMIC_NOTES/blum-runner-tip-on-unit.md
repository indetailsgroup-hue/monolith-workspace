# Blum TIP-ON and TIP-ON BLUMOTION Opening Support Units

> draft/review_ready — ชุดอุปกรณ์กดกระเด้งกลไกผ่านแกนเชื่อมซิงโครไนซ์ และระบบเปิดแบบกดกระเด้งพร้อมปิดนุ่มนวลร่วมกัน

## สรุป
ชุดอุปกรณ์สำหรับเสริมกลไกการเปิดด้วยแรงกดกลไก (TIP-ON และ TIP-ON BLUMOTION) สำหรับตระกูลรางซ่อนใต้ MOVENTO และ TANDEM รองรับหน้าบานที่กว้างและสูงเพื่อการเปิดที่นุ่มนวลและสม่ำเสมอ
- **TIP-ON for MOVENTO Pinion Set (ชุดเกียร์เฟืองหมุนประสาน):** รหัส `T57.7400.01` (สีเทาฝุ่น RAL 7037)
- **TIP-ON Synchronisation Linkage (แกนหมุนซิงโครไนซ์):** 
  - รุ่นกลม (MOVENTO): รหัส `ZST.1160W` (ความยาวอลูมิเนียมยึดพิน สำหรับตัด)
  - รุ่นโปรไฟล์เทา (TANDEM): รหัส `T57.1140S` (สำหรับตัดตามขนาด)
- **TIP-ON BLUMOTION Sets (สำหรับราง MOVENTO BLUMOTION S):** รหัส `T60Lxxxx` (แยกตามระดับความลึกช่วง NL และระดับน้ำหนักกล่องลิ้นชักรวม)
