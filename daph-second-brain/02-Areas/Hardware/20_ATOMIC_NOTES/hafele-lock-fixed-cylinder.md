# Häfele Furniture Locks with Fixed Cylinder Cores

> draft/review_ready — กลุ่มผลิตภัณฑ์ล็อคเฟอร์นิเจอร์แบบมีเสื้อแกนกุญแจตายตัว (Fixed Cylinder Core) ในตัว ประกอบด้วยแกน Ø18 mm และ Ø22 mm

## สรุป
กลุ่มกุญแจล็อคตู้เฟอร์นิเจอร์และลิ้นชักของ **Häfele** แบบติดไส้กุญแจตายตัวประกอบเสร็จมาเป็นชิ้นเดียวจากโรงงาน (Fixed Cylinder) เหมาะกับการติดตั้งระดับโรงงานและงานตู้ที่ไม่ต้องการเปลี่ยนผังรหัสแกนกุญแจบ่อย กลุ่มหลักใช้งานเจาะแกนผ่านไม้ขนาด **Ø18 mm** และรุ่นตลับบานเปิดสูง/ตลับขยายใช้แกนขนาด **Ø22 mm**

## ค่าเจาะ (drillMap-critical — pending engineer sign-off)

1. **ขนาดรูเจาะเสื้อกุญแจ (Cylinder Boring Diameter):**
   - รุ่นประดับไส้ขนาดเล็กหรือประเภทสวิตช์กดทั่วไปต้องการขนาดรูเจาะ **$\varnothing18\text{ mm}$**
   - รุ่นตลับบานประตูใหญ่หรือประเภทขยายพิน (Espagnolette Standard-Nova / Extending Rod) ต้องการขนาดรูเจาะ **$\varnothing22\text{ mm}$**
   - บานเลื่อนกระจกไม่ต้องเจาะบานกระจก แต่ขันน็อตประคองที่บานกระจกเปลือยแทน
2. **ระยะ Backset D Spacing:**
   - ปรับตามรุ่นประเภทสินค้า:
     - **$D = 25\text{ mm}$** (Deadbolt standard, Deadbolt Rim brass)
     - **$D = 24.5\text{ mm}$** (Deadbolt Standard plate 200, Roller Shutter Rim)
     - **$D = 22\text{ mm}$** (Mortice lock case D=22mm, สำหรับหน้าลิ้นชักลดเหลือ **$19\text{ mm}$**)
     - **$D = 20\text{ mm}$** (Inlaid lock, Rondell Flap)
     - **$D = 14\text{ mm}$** (Extending Rod lock)
     - **$D = 15 - 40\text{ mm}$** (Adjustable Deadbolt D)
3. **ระยะความลึกรูเจาะบานตู้ (Boring Depth):**
   - ความลึกการเจาะทะลุหน้าบานตู้สัมพันธ์ตามความยาวเสื้อสูบกุญแจ ตั้งแต่ **21 mm, 22 mm, 23.5 mm** ไปจนถึง **25 mm** ซึ่งวิศวกรออกแบบตู้จำเป็นต้องยืนยันความกว้างบานไม้และพิกัดความลึกอีกครั้งก่อนการเจาะจริง
