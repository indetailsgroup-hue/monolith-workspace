# Häfele — Rafix 20 & Rafix 30 Connector Systems

ระบบข้อต่อประกอบตู้และแผ่นชั้น (Rafix System) เจาะรูขนาดถ้วย **Ø20 mm** หรือ **Ø30 mm** บนแผ่นบอร์ด มีคุณสมบัติเด่นคือเป็นมุมขันเอียงทำให้สามารถขันยึดได้จากด้านบนและยึดแผ่นชั้นป้องกันการกระดกได้ดี

---

## 1. Rafix 20 Connector Housings (Ø20mm)

### 1.1 Rafix 20 without Dowel (แบบไม่มีเดือยยึดบอร์ด)
*เจาะรูถ้วยขนาด Ø20 mm ระยะกึ่งกลางรูถึงขอบบอร์ด Dim. A = 8.0 mm หรือ 9.5 mm ขึ้นกับความหนาไม้*
- **Zinc alloy (ตัวเรือนซิงค์อัลลอย, ขันยึดด้วย PZ2)**
  - ไม้หนา **16 mm**: ความลึกเจาะ D = **12.7 +0.2 mm**, Dim. A = **8.0 mm** (พาร์ต: Nickel ชนิดมีปีก `263.15.703` / ไม่มีปีก `263.11.703`)
  - ไม้หนา **19 mm**: ความลึกเจาะ D = **14.2 +0.2 mm**, Dim. A = **9.5 mm** (พาร์ต: Nickel ชนิดมีปีก `263.15.705` / ไม่มีปีก `263.11.705`)
- **Plastic (ตัวเรือนพลาสติก, ขันยึดด้วย PZ2)**
  - ไม้หนา **16 mm**: D = **12.7 +0.2 mm**, Dim. A = **8.0 mm** (พาร์ต: ขาว `263.10.703` / ดำ `263.10.303` / เทา `263.10.203`)
  - ไม้หนา **19 mm**: D = **14.2 +0.2 mm**, Dim. A = **9.5 mm** (พาร์ต: ขาว `263.10.705` / ดำ `263.10.305` / เทา `263.10.205`)

### 1.2 Rafix 20 for Flush Mounting (แบบเจาะจมเสมอผิวบอร์ด)
*ตัวเรือนพลาสติก ขอบบ่าเทเปอร์ (Countersunk rim)*
- ไม้หนา **16 mm**: D = **12.7 +0.2 mm**, Dim. A = **8.0 mm** (พาร์ต: ขาว `263.50.703` / น้ำตาล `263.50.103`)
- ไม้หนา **19 mm**: D = **14.2 +0.2 mm**, Dim. A = **9.5 mm** (พาร์ต: ขาว `263.50.705` / ดำ `263.50.305` / เบจ `263.50.405`)

### 1.3 Rafix 20 with Dowel (แบบมีเดือยยึดบอร์ดกันบิดหมุน)
- ไม้หนา **16 mm**: D = **12.7 +0.2 mm**, Dim. A = **8.0 mm** (พาร์ต: Zinc alloy Nickel `263.13.703` / พลาสติกขาว `263.12.703`)
- ไม้หนา **19 mm**: D = **14.2 +0.2 mm**, Dim. A = **9.5 mm** (พาร์ต: Zinc alloy Nickel `263.13.705` / พลาสติกขาว `263.12.705`)

### 1.4 Rafix Tab 20 (แบบดันสไลด์ประกอบ/ไม่ต้องขันสกรู)
*ตัวเรือนพลาสติกสีเทา มีฝาปิดสไลด์ในตัว (Sliding cover cap) สำหรับยึดชั้นแบบ Tool-less*
- ความลึกรูเจาะบอร์ด D = **14.2 +0.2 mm** (พาร์ต: `263.09.795`)
- ยึดคู่กับหมุดรับชั้นแผงข้าง ขนาดรูเจาะนำ Ø5 mm (พาร์ต: Zinc alloy `282.43.905` / Nickel `282.43.727`)

---

## 2. Rafix 30 Connector Housings (Ø30mm)
*ตัวเรือนพลาสติก มีปีกครอบ ขันแน่นด้วยไขควงปากแบน*
- ไม้หนาตั้งแต่ **16 mm** ขึ้นไป: ความลึกรูเจาะบอร์ด D = **13.0 mm**, Dim. A = **8.0 mm** (พาร์ต: ชนิดมีบ่ากันบิด `263.46.935` / ไม่มีบ่า `263.46.915`)

---

## 3. Connecting Bolts for Rafix

### 3.1 Rafix S20 Bolts (เกลียวพิเศษยึดไม้โดยตรง ขนาดรูนำ Ø5 mm)
*ใช้กับบอร์ดข้างตู้ทั่วไป*
- Thread length 11 mm (พาร์ต: Bright `263.20.141` / Galvanized `263.20.847`)
- Thread length 15 mm (พาร์ต: Bright `263.20.150` / Galvanized `263.20.856`)

### 3.2 Rafix M20 Bolts (เกลียวมิล M6/เหล็กและซิงค์อัลลอย)
- **Steel (เกลียวยาว 12 mm)**: พาร์ต Galvanized `263.20.981` (เหมาะสำหรับเกลียวปล่อยในไม้เนื้อแข็งหรือฝัง sleeve)
- **Zinc alloy (เกลียวยาว 7.5 mm)**: พาร์ต Bright `263.20.131`

### 3.3 Rafix S30 Bolts (สำหรับ Rafix 30)
- เกลียวหยาบพิเศษยาว 12 mm สำหรับรูนำ Ø5 mm (พาร์ต Galvanized `263.48.904`)

### 3.4 Double-ended Bolts (แกนยึดยึดบอร์ดสองฝั่ง/แผงกลางตู้)
- **2-piece สำหรับแผงกลาง 16–22 mm**: แกนเสียบเกลียวขันล็อคประกบกันเองตรงกลาง รูเจาะผ่านแผง Ø5 mm (พาร์ต Galvanized `263.24.943`)
- **1-piece สำหรับแผงกลาง 19 mm**: แกนชิ้นเดียวล็อค snap ring รูเจาะผ่านแผง Ø8 mm (พาร์ต Galvanized `263.24.934`)
