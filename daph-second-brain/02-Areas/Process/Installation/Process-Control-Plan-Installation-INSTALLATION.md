# Process-Control-Plan-Installation-INSTALLATION

## สรุป

เอกสารประเภท **Process Control Plan** ในกลุ่มกระบวนการ **Installation** ครอบคลุมหน่วยกระบวนการ: การบรีฟงาน, การตรวจสอบหน้างาน, การตรวจสอบระยะ, การปูรองพื้น, เรียงอุปกรณ์, การติดตั้งโครงอลูมิเนียม, การตรวจสอบขนาดตู้, การจัดวางตู้, การติดตั้งผนัง, การติดตั้งท๊อป, การติดตั้งอุปกรณ์ภายในตู้, งานระบบไฟฟ้า, งานเก็บซิลิโคน, การตรวจสอบหน้าบาน, การรักษาความสะอาด, การเก็บของ
