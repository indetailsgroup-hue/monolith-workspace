# Furniture Deep Research — ค้นคว้าข้อมูลเฟอร์นิเจอร์เชิงลึก

## ภาพรวม

Skill นี้ช่วยค้นคว้าข้อมูลเชิงลึกเกี่ยวกับเฟอร์นิเจอร์ทุกแง่มุม ตั้งแต่การหาอุปกรณ์ เปรียบเทียบสินค้า หาราคา ไปจนถึงการวิเคราะห์แนวโน้มตลาดและการออกแบบ

เมื่อผู้ใช้ถามคำถามเกี่ยวกับเฟอร์นิเจอร์ที่ต้องการข้อมูลเชิงลึก ให้ทำตามขั้นตอนด้านล่าง

---

## ขั้นตอนการค้นคว้า

### Step 1: วิเคราะห์คำถาม

จำแนกประเภทคำถามก่อน:

| ประเภท | ตัวอย่างคำถาม | แหล่งข้อมูลหลัก |
|--------|-------------|---------------|
| **อุปกรณ์/ฮาร์ดแวร์** | "Minifix ขนาดไหนเหมาะกับแผ่น 18mm" | Skill furniture-hardware + Web |
| **ราคา/ซัพพลายเออร์** | "ราคาบานพับ Blum CLIP top ที่ไหนถูก" | Web search (Shopee, Lazada, ตัวแทน) |
| **เปรียบเทียบสินค้า** | "Blum vs Hettich บานพับตัวไหนดีกว่า" | Web + furniture-hardware |
| **การออกแบบ** | "แนวทางออกแบบตู้ครัว modern" | Web search (Pinterest, Houzz) |
| **วัสดุ** | "MDF กับ Plywood ตัวไหนเหมาะกับตู้ห้องน้ำ" | furniture-hardware + Web |
| **แนวโน้มตลาด** | "เทรนด์เฟอร์นิเจอร์ปี 2026" | Web search (trade publications) |
| **วิธีทำ/DIY** | "วิธีทำตู้เสื้อผ้า built-in" | furniture-hardware + Web |

### Step 2: ค้นคว้าข้อมูล

**กฎสำคัญ**: ค้นคว้าจากหลายแหล่งเสมอ อย่าพึ่งแหล่งเดียว

#### แหล่งข้อมูลตามประเภท

**สำหรับข้อมูลทางเทคนิค (สเปค, ขนาด, วิธีติดตั้ง)**:
1. ดูจาก Skill `furniture-hardware` ก่อน (ถ้ามี)
2. ค้นหาเพิ่มเติมจากเว็บ:
   - Häfele catalog (hafele.com)
   - Blum product data (blum.com)
   - Hettich product data (hettich.com)
   - Grass product data (grass.eu)

**สำหรับข้อมูลราคาและแหล่งซื้อ (ประเทศไทย)**:
1. ค้นหาจาก:
   - Shopee Thailand
   - Lazada Thailand
   - Häfele Thailand (hafele.co.th)
   - Homepro (homepro.co.th)
   - Thai Watsadu (thaiwatsadu.com)
   - Global House
   - ร้านค้าเฉพาะทาง (ย่านวังบูรพา, ถ.เจริญรัถ)

**สำหรับการออกแบบและแรงบันดาลใจ**:
1. ค้นหาจาก:
   - Pinterest
   - Houzz
   - ArchDaily
   - Dezeen
   - Instagram (hashtags: #kitchendesign, #furnituredesign)

**สำหรับแนวโน้มตลาด**:
1. ค้นหาจาก:
   - CSIL (Centre for Industrial Studies)
   - Furniture Today
   - Wood & Panel magazines
   - TIFF (Thailand International Furniture Fair)

### Step 3: รวบรวมและวิเคราะห์

เมื่อได้ข้อมูลมาแล้ว ให้จัดเรียงเป็นโครงสร้างที่ชัดเจน:

#### สำหรับการเปรียบเทียบสินค้า:
```
## [ชื่อสินค้า A] vs [ชื่อสินค้า B]

### ตารางเปรียบเทียบ
| หัวข้อ | สินค้า A | สินค้า B |
|-------|---------|---------|
| ราคา | ... | ... |
| คุณสมบัติ | ... | ... |
| ข้อดี | ... | ... |
| ข้อเสีย | ... | ... |

### สรุปและคำแนะนำ
- เลือก A ถ้า...
- เลือก B ถ้า...
```

#### สำหรับรายงานวิจัย:
```
## [หัวข้อวิจัย]

### บทสรุปผู้บริหาร
(สรุปสั้นๆ 3-5 บรรทัด)

### ข้อค้นพบหลัก
1. ...
2. ...
3. ...

### รายละเอียด
(เนื้อหาเชิงลึก)

### แหล่งอ้างอิง
- [ชื่อแหล่ง](URL)
```

#### สำหรับคำแนะนำอุปกรณ์:
```
## คำแนะนำ: [หัวข้อ]

### ความต้องการ
- [สรุปจากคำถามผู้ใช้]

### ตัวเลือกที่แนะนำ
1. **[ตัวเลือก A]** — ราคา: ฿xxx
   - ข้อดี: ...
   - ข้อเสีย: ...
2. **[ตัวเลือก B]** — ราคา: ฿xxx
   - ข้อดี: ...
   - ข้อเสีย: ...

### คำแนะนำ
เลือก [X] เพราะ...

### แหล่งซื้อ
- [ร้าน/เว็บ](URL) — ราคา ฿xxx
```

### Step 4: ส่งมอบ

**รูปแบบการส่งมอบ**:
- ถ้าข้อมูลสั้น → ตอบในแชทเลย
- ถ้าข้อมูลยาว (>500 คำ) → สร้างเป็นไฟล์ .md หรือ .docx
- ถ้ามีตัวเลข/ราคาเยอะ → พิจารณาสร้าง .xlsx เพิ่ม

**กฎการอ้างอิง**:
- ระบุแหล่งที่มาเสมอ
- ถ้าเป็นราคา ให้ระบุวันที่ค้นหาด้วย (ราคาเปลี่ยนแปลงได้)
- ถ้าเป็นสเปคทางเทคนิค ให้ระบุเอกสารอ้างอิง (catalog, datasheet)

## เทมเพลตคำค้นหา

เมื่อต้อง search เว็บ ใช้คำค้นเหล่านี้เป็นแนวทาง:

**หาสเปค**: `"[product name]" specifications datasheet filetype:pdf`
**หาราคาไทย**: `"[product name]" ราคา site:shopee.co.th OR site:lazada.co.th`
**หาตัวแทน**: `"[brand]" distributor Thailand`
**หาคู่มือ**: `"[product]" installation guide instructions`
**เปรียบเทียบ**: `"[product A]" vs "[product B]" comparison review`
**เทรนด์**: `furniture trend [year] [style/material]`
