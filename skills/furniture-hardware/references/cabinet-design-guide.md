## 1. มุมมองตู้ (Cabinet Views)

การอ่านแบบเฟอร์นิเจอร์ต้องเข้าใจมุมมองทั้ง 3 ด้านหลัก:

### 1.1 Front View (มองจากด้านหน้า)

```
        ┌─────────────────────────┐
        │       Top Panel         │
        ├─────────────────────────┤
        │  │                   │  │
        │  │                   │  │
        │ L│    ช่องว่างตู้     │R │
        │ e│   (Cabinet space)  │i │
        │ f│                   │g │
        │ t│                   │h │
        │  │                   │t │
        │  │                   │  │
        ├─────────────────────────┤
        │      Bottom Panel       │
        └─────────────────────────┘
              ↑ Front (หน้าตู้)
```

**สิ่งที่เห็นใน Front View:**
- ตำแหน่ง Hinge (บานพับ) ที่ขอบซ้าย/ขวาของบาน
- ตำแหน่ง Minifix bolt (มองเห็นเป็นจุดที่ขอบแผ่นตั้ง)
- ขนาดบานประตู เทียบกับขนาดตัวตู้

### 1.2 Top View (มองจากด้านบน)

```
                    Black (หลังตู้)
                        ↓
        ┌─────────────────────────┐
        │                         │
        │  Minifix ●──── Dowel ■  │ ← Left Panel
        │                         │
        │       Inset Top         │
        │    (แผ่นบน/ล่าง)        │
        │                         │
        │  Dowel ■────● Minifix   │ ← Right Panel
        │                         │
        └─────────────────────────┘
                        ↑
                    Front (หน้าตู้)
```

**สิ่งที่เห็นใน Top View:**
- ตำแหน่ง Minifix cam housing (Ø15) บนแผ่นนอน (Top/Bottom)
- ทิศทาง Bolt ที่ชี้เข้าหาแผ่นตั้ง (Left/Right panel)
- ตำแหน่ง Dowel ที่อยู่ระหว่าง Minifix
- ทิศทาง Front/Back ของตู้

### 1.3 Side View (มองจากด้านข้าง)

```
        ┌─────────────────────────┐
        │       Top Panel         │
        │ ●Minifix  ■Dowel       │
        │                         │
        │                         │
        │       Side Panel        │
        │      (แผ่นข้าง)         │
        │                         │
        │                         │
        │ ■Dowel   ●Minifix      │
        │      Bottom Panel       │
        └─────────────────────────┘
        ← Back              Front →
```

**สิ่งที่เห็นใน Side View:**
- ตำแหน่ง Minifix + Dowel ที่ยึดแผ่นบน/ล่างเข้ากับแผ่นข้าง
- ระยะห่างจากขอบหน้าและขอบหลัง
- ระบบรู System 32 (ถ้ามี)

## 2. ประเภทบานตู้ (Door Types)

### 2.1 Overlay (บานปิดทับ)

```
     ┌───────────────────────────────┐
     │  ┌──H──────────────────H──┐   │
     │  │                        │   │
     │  │      บานประตู          │   │
     │  │    (Door Panel)        │   │
     │  │                        │   │
     │  │  บานทับขอบเสาตู้       │   │
     │  │  (ปิดคลุมตัวตู้)       │   │
     │  │                        │   │
     │  │                        │   │
     │  └──H──────────────────H──┘   │
     └───────────────────────────────┘
     H = Hinge (บานพับ)
```

**ลักษณะ:**
- บานประตูมีขนาด **ใหญ่กว่า** ช่องเปิดตู้
- บานปิดทับเสาตู้ทั้ง 2 ข้าง (Full overlay) หรือ 1 ข้าง (Half overlay)
- **Hinge**: ใช้ Full overlay hinge (Overlay 16-19mm ขึ้นกับยี่ห้อ)
- **ระยะ Overlay**: ปกติ = ความหนาแผ่น - 3mm (เช่น แผ่น 18mm → overlay 15mm)

**ตำแหน่ง Hinge สำหรับ Overlay:**
- บานพับอยู่บนเสาตู้ (Side panel)
- Cup Ø35mm เจาะที่บาน ห่างจากขอบ **4-5mm**
- Mounting plate ยึดบนเสา

### 2.2 Inset (บานฝังใน)

```
     ┌──────────────────────────────┐
     │         Top Panel            │
     │  ┌─H────────────────H─┐     │
     │  │                    │     │
     │  │    บานประตู        │     │
     │  │  (Door Panel)      │     │
     │  │                    │     │
     │  │  บานอยู่ภายใน      │     │
     │  │  ระนาบเดียวกับ     │     │
     │  │  หน้าตู้            │     │
     │  │                    │     │
     │  └─H────────────────H─┘     │
     │        Bottom Panel          │
     └──────────────────────────────┘
     H = Hinge (บานพับ)
```

**ลักษณะ:**
- บานประตูมีขนาด **เล็กกว่า** ช่องเปิดตู้ (ลบ gap 2-3mm ทุกด้าน)
- บานอยู่ในระนาบเดียวกับหน้าเสาตู้ (Flush)
- **Hinge**: ใช้ Inset hinge (0mm overlay / negative overlay)
- งาน Premium / ดีไซน์สูง

**ตำแหน่ง Hinge สำหรับ Inset:**
- Cup Ø35mm เจาะที่บาน
- Mounting plate ยึดบนเสา ด้านใน

### 2.3 ผลต่อตำแหน่ง Minifix

**Overlay Cabinet:**
- Minifix bolt ชี้เข้าหาเสาตู้ปกติ
- ระยะจาก Bolt center ถึงหน้าแผ่น = ความหนา/2 (เช่น แผ่น 18mm → 9mm)
- แผ่นบน/ล่างอยู่ **ระหว่าง** เสาซ้าย-ขวา (ส่วนใหญ่)

**Inset Cabinet:**
- Minifix ต้องคำนึงถึงระยะ Inset ของบาน
- แผ่นบน/ล่างอาจ **ยื่นออก** มาเท่ากับระยะ Overlay ที่เหลือ
- ระยะ Minifix จากขอบหน้า (Front edge) อาจต่างจาก Overlay
- ตัวอย่างจากแบบ: L 11 (ระยะ Minifix จากขอบ), B 24 (Bolt 24mm)

### 2.4 ตารางเปรียบเทียบ

| รายการ | Overlay | Inset |
|--------|---------|-------|
| ขนาดบาน | ใหญ่กว่าช่องเปิด | เล็กกว่าช่องเปิด |
| Hinge type | Full/Half overlay | Inset (0mm) |
| ความยาก | ง่ายกว่า | ยากกว่า (ต้องแม่นยำ) |
| Gap ที่เห็น | ไม่เห็น (บานปิดทับ) | เห็น gap 2-3mm รอบบาน |
| ราคา | ถูกกว่า | แพงกว่า |
| Minifix placement | มาตรฐาน | ต้องปรับตาม inset depth |
| ดีไซน์ | ทั่วไป | Premium, high-end |

## 3. การวาง Minifix + Dowel Layout

### 3.1 หลักการพื้นฐาน

Minifix และ Dowel ใช้ร่วมกันเพื่อยึดแผ่นตั้ง (Side panel) กับแผ่นนอน (Top/Bottom panel):

- **Minifix** = ตัวล็อกหลัก ให้แรงยึด (ถอดประกอบได้)
- **Dowel** = ตัวจัดตำแหน่ง ป้องกันแผ่นเลื่อน

### 3.2 กฎการจัดวาง (Layout Rules)

#### ตามความยาวแผ่น (Panel Length)

| ความยาวแผ่น | Minifix | Dowel | รูปแบบการจัดเรียง |
|-------------|---------|-------|------------------|
| < 300mm | 1 | 1 | Minifix - Dowel |
| 300-600mm | 2 | 1 | Minifix - Dowel - Minifix |
| 600-900mm | 2 | 2 | Minifix - Dowel - Dowel - Minifix |
| 900-1200mm | 3 | 2 | Minifix - Dowel - Minifix - Dowel - Minifix |
| > 1200mm | 3+ | 3+ | เพิ่มตามอัตราส่วน |

#### ตัวอย่างจริง: แผ่น 500mm (จากแบบ)

```
   37  ←──→  Minifix ●
              |
   32         Dowel ■
              |
   32         |
              |  ← 500mm (ความยาวรวม)
   32         |
              |
   32         Dowel ■
              |
   37  ←──→  Minifix ●

   ระยะ System 32: 37 + 32×(n) + 37
```

จากแบบที่เห็น (Inset Top view):
- **จุดบน**: Minifix + Dowel (ห่างจากขอบบน 37mm)
- **จุดกลาง**: Dowel + Minifix + Dowel (กลางแผ่น)
- **จุดล่าง**: Dowel + Minifix (ห่างจากขอบล่าง 37mm)
- **ระยะห่างแต่ละจุด**: 32mm (System 32)

### 3.3 ระยะห่างจากขอบ (Edge Distances)

```
  ←── 30mm ──→ ←── 11mm ──→
  ┌───────────┬──────────────┐
  │           │              │
  │  Minifix  │   Dowel      │  ← ห่างจากขอบหน้า (Front edge)
  │  center   │   center     │
  │           │              │
  └───────────┴──────────────┘
       ↕ 19mm (ความหนาแผ่น)
```

| ระยะ | ค่า | คำอธิบาย |
|------|-----|---------|
| Minifix center จากขอบหน้า | **37mm** | = ระยะ System 32 แรก |
| Minifix center จากขอบหลัง | **37mm** | = ระยะ System 32 สุดท้าย |
| Minifix center จากผิวแผ่น | **9-9.5mm** | = ครึ่งหนึ่งของความหนาแผ่น |
| Bolt center ถึงผิวแผ่นตั้ง | **30mm** (B24) หรือ **37mm** (B34) | ขึ้นกับความยาว bolt |
| Dowel center จากขอบ | **11mm** | มาตรฐานทั่วไป |
| ระยะ Minifix-Dowel ถัดไป | **32mm** | System 32 |

### 3.4 มิติจากแบบจริง (Inset Top View)

จากรูปที่ให้มา:
- **L 11** = ระยะจากขอบแผ่นนอนถึง center Minifix/Dowel = 11mm
- **B 24** = ความยาว Bolt = 24mm
- **30** = ระยะจาก center bolt ถึงขอบแผ่นตั้ง = 30mm
- **11** = ระยะจากผิวแผ่นนอนถึง center cam = 11mm (Minifix 12) หรือ ระยะ dowel
- **19** = ความหนาแผ่นนอน = 19mm

## 4. Minifix Transform — กฎการหมุน/พลิก

เมื่อวาง Minifix ในซอฟต์แวร์ CAD จำเป็นต้องปรับทิศทางให้ถูกต้องตามมุมที่ติดตั้ง

### 4.1 องค์ประกอบ Minifix Transform

| ฟังก์ชัน | คีย์ลัด | คำอธิบาย |
|---------|---------|---------|
| **Vertical Flip** | V | พลิกแนวตั้ง — เปลี่ยนทิศ bolt จากบน↔ล่าง |
| **Horizontal Flip** | H | พลิกแนวนอน — เปลี่ยนทิศ bolt จากซ้าย↔ขวา |
| **Rot X** | ±15° | หมุนรอบแกน X (fine-tune) |
| **Rot Y** | ±15° | หมุนรอบแกน Y (fine-tune) |
| **Rot Z** | ±15° | หมุนรอบแกน Z (fine-tune) |
| **Move X** (L/R) | -5 ถึง +5 | เลื่อนซ้าย/ขวา |
| **Move Y** (U/D) | -5 ถึง +5 | เลื่อนขึ้น/ลง |
| **Move Z** (F/B) | -5 ถึง +5 | เลื่อนหน้า/หลัง |

### 4.2 กฎ Corner — วิธีเลือก Flip ตามมุมตู้

มุมที่ Minifix อยู่กำหนดว่าต้อง Flip อย่างไร:

```
     ┌──────────────────────────────────┐
     │                                  │
     │  TOP LEFT          TOP RIGHT     │
     │  V: OFF            V: OFF        │
     │  H: OFF            H: ON         │
     │                                  │
     │                                  │
     │                                  │
     │                                  │
     │  BOTTOM LEFT       BOTTOM RIGHT  │
     │  V: ON             V: ON         │
     │  H: OFF            H: ON         │
     │                                  │
     └──────────────────────────────────┘
                   Front View
```

#### ตารางสรุป Flip ตามมุม (Front View)

| มุม (Corner) | Vertical Flip | Horizontal Flip | Bolt ชี้ไปทาง |
|-------------|---------------|-----------------|--------------|
| **TOP LEFT** | OFF | OFF | ลง + ขวา (→เข้าเสาซ้าย) |
| **TOP RIGHT** | OFF | ON | ลง + ซ้าย (→เข้าเสาขวา) |
| **BOTTOM LEFT** | ON | OFF | ขึ้น + ขวา (→เข้าเสาซ้าย) |
| **BOTTOM RIGHT** | ON | ON | ขึ้น + ซ้าย (→เข้าเสาขวา) |

#### หลักการจำง่าย:
- **Vertical Flip (V)** = เปิด เมื่อ Minifix อยู่ **ด้านล่าง** (BOTTOM) — bolt ต้องชี้ขึ้น
- **Horizontal Flip (H)** = เปิด เมื่อ Minifix อยู่ **ด้านขวา** (RIGHT) — bolt ต้องกลับด้าน

### 4.3 กฎ Corner — Top View (มองจากบน)

```
                    Back (หลังตู้)
        ┌───────────────────────────────┐
        │                               │
        │  BACK LEFT      BACK RIGHT    │
        │  V: -           V: -          │
        │  H: OFF         H: ON         │
        │                               │
        │                               │
        │  FRONT LEFT     FRONT RIGHT   │
        │  V: -           V: -          │
        │  H: OFF         H: ON         │
        │                               │
        └───────────────────────────────┘
                    Front (หน้าตู้)
```

**หมายเหตุ:**
- ใน Top View, Vertical Flip มักไม่เปลี่ยน (เพราะมิติบน-ล่างคือความหนาแผ่น)
- **H Flip** เปลี่ยนตามฝั่งซ้าย/ขวา เช่นเดียวกับ Front View

### 4.4 Fine Rotation (Rot X/Y/Z)

ใช้สำหรับ **การปรับละเอียด** เมื่อ Minifix ไม่ได้อยู่ในแนวมาตรฐาน:

| สถานการณ์ | ปรับแกน | ค่า |
|----------|---------|-----|
| เจาะแบบเอียง (Angled joint) | Rot Z | ±15° ตามมุมเจาะ |
| แผ่นตั้งเอียง (Tapered panel) | Rot Y | ±15° |
| ปรับ Cam housing ให้หมุนถูกทิศ | Rot X | ±15° |

### 4.5 Move Hardware (เลื่อนตำแหน่ง)

| แกน | ทิศทาง | ใช้เมื่อ |
|-----|--------|---------|
| X (L/R) | ซ้าย-ขวา | เลื่อนให้ตรงกับ System 32 |
| Y (U/D) | ขึ้น-ลง | ปรับระยะจากผิวแผ่น |
| Z (F/B) | หน้า-หลัง | ปรับระยะจากขอบหน้า/หลัง |

**"Clamped to cabinet bounds"** = ระบบจำกัดไม่ให้ hardware เลื่อนออกนอกขอบแผ่น

## 5. System 32 — ระบบเจาะมาตรฐาน

### 5.1 หลักการ System 32

System 32 (หรือ 32mm System) เป็นมาตรฐานสากลสำหรับการเจาะรูเฟอร์นิเจอร์:

```
    ← 37mm →          ← 37mm →
    ┌────────────────────────────┐
    │  ●                     ●  │  ← 37mm จากขอบบน
    │  │                     │  │
    │  │  32mm               │  │
    │  │                     │  │
    │  ●                     ●  │
    │  │                     │  │
    │  │  32mm               │  │
    │  │                     │  │
    │  ●                     ●  │  ← ซ้ำทุก 32mm
    │  │                     │  │
    │  │  32mm               │  │
    │  │                     │  │
    │  ●                     ●  │
    │  │                     │  │
    │  │  32mm               │  │
    │  │                     │  │
    │  ●                     ●  │  ← 37mm จากขอบล่าง
    └────────────────────────────┘
```

### 5.2 ระยะสำคัญ

| พารามิเตอร์ | ค่า | หมายเหตุ |
|-----------|-----|---------|
| ระยะห่างระหว่างรู | **32mm** | มาตรฐานสากล |
| ระยะจากขอบหน้า (Front) | **37mm** | แถวหน้า |
| ระยะจากขอบหลัง (Back) | **37mm** | แถวหลัง (ถ้ามี) |
| ระยะจากขอบบน/ล่าง | **37mm** | รูแรก/สุดท้าย |
| ขนาดรู Shelf pin | **Ø5mm** | ลึก 13mm |
| ขนาดรู Minifix bolt | **Ø5mm** หรือ **Ø8mm** | ขึ้นกับ bolt type |
| ขนาดรู Hinge plate | **Ø3mm** | pilot hole |

### 5.3 ตัวอย่างการคำนวณ (จากแบบ 500mm)

แผ่นข้างสูง 500mm:
```
ระยะจากขอบบน:    37mm  → จุดที่ 1 (Minifix/Hinge plate)
                 +32mm → จุดที่ 2 (Shelf pin / Dowel)
                 +32mm → จุดที่ 3
                 +32mm → จุดที่ 4
                 ...
                 +32mm → จุดที่ n
ระยะจากขอบล่าง:  37mm  → จุดสุดท้าย

จำนวนช่วง = (500 - 37 - 37) / 32 = 426 / 32 = 13.3 → 13 ช่วง
จำนวนรู = 13 + 1 = 14 รู ต่อแถว
```

### 5.4 ความสัมพันธ์ System 32 กับ Hardware

| Hardware | ตำแหน่งบน System 32 |
|----------|-------------------|
| Minifix cam | ใช้รู System 32 สำหรับ bolt hole (Ø5mm) |
| Shelf pin | ใช้รู System 32 โดยตรง (Ø5mm) |
| Hinge mounting plate | ยึดที่ตำแหน่ง System 32 |
| Dowel | วางระหว่างจุด System 32 (ห่าง 32mm) |
| Drawer slide | ยึดที่ตำแหน่ง System 32 |

├──┤                    ├──┤
│  │                    │  │
│  │●Mfx             Mfx●│  │  ← Shelf pin rows
│  │■Dwl             Dwl■│  │

## 7. สรุป Checklist การวาง Hardware

### ก่อนวาง Minifix:
- [ ] กำหนด Door type: Overlay หรือ Inset?
- [ ] กำหนดความหนาแผ่น: 16 / 18 / 19 / 25mm?
- [ ] เลือก Bolt: B24 (แผ่นบาง) หรือ B34 (แผ่นหนา)?
- [ ] กำหนดระยะ System 32: 37mm จากขอบ?

### การวาง:
- [ ] เลือก Corner ถูกต้อง (TOP LEFT/RIGHT, BOTTOM LEFT/RIGHT)
- [ ] ตั้ง Vertical Flip: ON ถ้าอยู่ BOTTOM
- [ ] ตั้ง Horizontal Flip: ON ถ้าอยู่ RIGHT
- [ ] จำนวน Minifix + Dowel เหมาะกับความยาวแผ่น
- [ ] Dowel อยู่ระหว่าง Minifix (ห่าง 32mm)

### ตรวจสอบ:
- [ ] Bolt ชี้เข้าหาแผ่นตั้ง (Side panel) ทุกจุด
- [ ] Cam housing อยู่บนแผ่นนอน (Top/Bottom) ทุกจุด
- [ ] ระยะจากขอบถูกต้อง (37mm / 30mm / 11mm)
- [ ] ไม่ชนกับ hardware อื่น (Hinge, Shelf pin, Drawer slide)
