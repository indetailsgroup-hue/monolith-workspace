## การติดตั้ง MINIFIX — Step by Step

### ขั้นตอนที่ 1: กำหนดจุดเจาะ

**แผ่นนอน (รับ Cam Housing)**:
```
ขอบแผ่น
  ↕ 8mm (กึ่งกลางแผ่น 16mm)
  ● รู Dowel Ø8mm ลึก 20mm
  ↕ ระยะห่าง ~32mm
  ● รู Dowel Ø8mm ลึก 20mm (ถ้ามี 2 dowel)

ด้านหน้าแผ่น (เจาะจากด้าน flat):
  ↕ 37mm จากขอบ (สำหรับ bolt 34mm)
  ● รู Cam Housing Ø15mm ลึก 14mm
```

**แผ่นตั้ง (รับ Bolt)**:
```
ขอบแผ่น (ด้านปลาย)
  ↕ 8mm จากขอบ (กึ่งกลางแผ่น 16mm)
  ● รู Dowel Ø8mm ลึก 20mm

ด้านหน้าแผ่น:
  ↕ 37mm จากขอบปลาย
  ● รู Bolt Ø5mm ทะลุ
  (ฝั่งด้าน) ขยายเป็น Ø8mm ลึก 8mm สำหรับหัว bolt
```

### ขั้นตอนที่ 2: เจาะรู
1. ยึดแผ่นให้แน่น ไม่ขยับ
2. เจาะรู Cam Housing (Ø15mm ลึก 14mm) ด้วย Forstner
3. เจาะรู Bolt (Ø5mm ทะลุ) ด้วยดอกเหล็ก
4. เจาะรู Dowel (Ø8mm ลึก 20mm) ทั้งสองแผ่น
5. **ทำความสะอาด**ขี้เลื่อยออกจากรูให้หมด

### ขั้นตอนที่ 3: ประกอบ
1. ตอก Dowel เข้าแผ่นตั้ง (ใช้ค้อนยาง)
2. ขัน Bolt เข้าแผ่นตั้ง (ให้หัว bolt โผล่ตามระยะ)
3. ดัน Cam Housing เข้ารูที่แผ่นนอน (press-fit)
4. สอดแผ่นตั้งเข้าแผ่นนอน (Dowel + Bolt เข้าตำแหน่ง)
5. ใช้ไขควง PZ2 หมุน Cam **ตามเข็มนาฬิกา** จนล็อก (90-180°)
6. ตรวจสอบความแน่น — แผ่นไม่ขยับ

### ข้อผิดพลาดที่พบบ่อย
- เจาะรู Cam ทะลุ → แผ่นเสีย ต้องเปลี่ยน
- ระยะ Cam-Bolt ไม่ตรง → ประกอบไม่เข้า
- ลืมทำความสะอาดรู → Cam ไม่เข้าสุด
- ใช้ดอก hole saw แทน Forstner → ก้นรูไม่เรียบ Cam หลุด

## การติดตั้งบานพับ CONCEALED HINGE

### ขั้นตอนที่ 1: เจาะรู Cup ที่บานประตู
```
ขอบบาน
  ← 4-5mm →
  ● Ø35mm ลึก 12mm (Forstner)

ตำแหน่งจากขอบบน/ล่าง:
  80-100mm จากขอบ (บานพับตัวบน/ล่าง)
```

### ขั้นตอนที่ 2: ยึด Mounting Plate ที่ตัวตู้
```
ขอบเสาตู้
  ← ระยะ overlay →
  สกรู 2 ตัว ระยะ 52mm

ตำแหน่ง: ตรงกับตำแหน่งบานพับที่บาน
```

### ขั้นตอนที่ 3: ประกอบและปรับ
1. สอด hinge cup เข้ารู → ขันสกรู 2 ตัว
2. คลิก hinge arm เข้า mounting plate
3. ปรับ 3 ทิศทาง:
   - **ซ้าย-ขวา**: สกรูตัวหน้า (ที่ arm)
   - **บน-ล่าง**: สกรู mounting plate (รูยาว)
   - **เข้า-ออก**: สกรูตัวหลัง (ที่ arm)

### จำนวนบานพับตามความสูงบาน
| ความสูงบาน | จำนวน | น้ำหนักสูงสุด |
|-----------|-------|-------------|
| ไม่เกิน 1000mm | 2 ตัว | 14kg |
| 1000-1500mm | 3 ตัว | 21kg |
| 1500-2000mm | 4 ตัว | 28kg |
| 2000-2500mm | 5 ตัว | 35kg |

## การติดตั้ง CONFIRMAT SCREW

### Step by Step
1. จัดแผ่นให้ตรง ใช้ฉากวัด 90°
2. หนีบแผ่นด้วยแคลมป์
3. เจาะรูนำด้วย Confirmat drill bit (เจาะทะลุแผ่นนอก เข้าเนื้อแผ่นตั้ง)
4. ขัน Confirmat เข้าด้วยดอก PZ2 หรือ PZ3
5. **อย่าขันแน่นเกินไป** — Particleboard จะบวม/แตกร่วน
6. ปิดหัวสกรูด้วยฝาครอบสี

### เคล็ดลับ
- เจาะรูนำเสมอ (ห้ามขันตรง!)
- ใช้ไขควงปรับแรงบิดได้ (torque limiter)
- ในไม้ MDF ให้เจาะรูนำเล็กกว่าปกติ 0.5mm

## การติดตั้งรางลิ้นชัก BALL BEARING

### ขั้นตอน
1. **ถอดราง**: แยก inner rail (ติดลิ้นชัก) กับ outer rail (ติดตู้)
2. **ยึด outer rail ที่ตัวตู้**:
   - วัดระยะจากขอบหน้าตู้ — ราง flush กับขอบ
   - ยึดด้วยสกรู 3.5x16mm (3-4 ตัว)
   - ใช้ระดับน้ำตรวจแนวราบ
3. **ยึด inner rail ที่ลิ้นชัก**:
   - ราง flush กับขอบหน้าลิ้นชัก
   - ยึดด้วยสกรู 3.5x16mm
4. **สอดลิ้นชัก**: ดัน inner rail เข้า outer rail จนคลิก
5. **ทดสอบ**: เลื่อนเข้า-ออก ต้องลื่นไม่สะดุด

### ขนาดช่องว่าง (Clearance)
- **ข้างละ**: 12.5-13mm (รวม 2 ข้าง = 25-26mm)
- **ตัวอย่าง**: ช่องตู้กว้าง 500mm → ลิ้นชักกว้าง 474-475mm

## System 32 — มาตรฐานการเจาะรู

### หลักการ
System 32 เป็นมาตรฐานอุตสาหกรรมสากลสำหรับการเจาะรูในตัวตู้เฟอร์นิเจอร์

### กฎหลัก
- **ระยะระหว่างรู**: 32mm (center to center)
- **ระยะจากขอบหน้า**: 37mm
- **ขนาดรู**: Ø5mm ลึก 13mm
- **แถวรู**: เจาะตามแนวตั้ง ทั้งฝั่งซ้ายและขวาของตัวตู้

### ประโยชน์
- รองรับอุปกรณ์ทุกยี่ห้อ (Blum, Hettich, Häfele, Grass)
- ปรับระดับชั้นวางได้ทุก 32mm
- ใช้เป็นจุดยึด mounting plate ของบานพับ
- ใช้ CNC drilling เจาะได้แม่นยำ

### Layout Template สำหรับตู้มาตรฐาน (สูง 720mm)
```
รูที่  | ระยะจากล่าง
------|------------
1     | 37mm
2     | 69mm
3     | 101mm
...   | (เพิ่มทีละ 32mm)
21    | 677mm
```
